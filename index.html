Use it on parent for all elements - just add transpie class!

<div class="transpie"> <input type="text" /> <input type="button" value="nice"/> </div>

or on one element!

<input type="text" class="transpie" /> <input type="button" value="default"/>

or call jquery function on (parent) element created dynamically (only needed for select, radio and checkbox)

HTML
<select class='transpie'><option>Dynamic</option><option>Dynamic2</option></select>
<input class='transpie' type='checkbox' />
<input class='transpie' type='radio' />

Function call
$(".transpie").cTP();
